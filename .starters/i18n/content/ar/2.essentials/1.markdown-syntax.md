## العناوين

استخدم العناوين لتقديم الأقسام الرئيسية. فهي تنظّم التوثيق وتساعد المستخدمين على التنقل داخل المحتوى.

::code-preview
---
class: "[&>div]:*:my-0"
---
## العناوين

#code
```mdc
## العناوين
```
::

### العناوين الفرعية

استخدم العناوين الفرعية لتقسيم الأقسام بشكل أدق. فهي تنشئ تسلسلًا هرميًا أوضح للمحتوى وتساعد على القراءة.

::code-preview
---
class: "[&>div]:*:my-0"
---
### العناوين الفرعية

#code
```mdc
### العناوين الفرعية
```
::

::tip
كل عنوان وعنوان فرعي ينشئ مرساة ويظهر تلقائيًا داخل جدول المحتويات.
::

## تنسيق النص

يدعم Docus أغلب خيارات تنسيق Markdown.

| النمط    | الصيغة      | النتيجة   |
| -------- | ------------ | ---------- |
| عريض     | `**عريض**`   | **عريض**   |
| مائل     | `*مائل*` | *مائل* |
| مشطوب    | `~~مشطوب~~`  | ~~مشطوب~~  |

ادمج الأنماط لإثراء النص وإبراز العناصر المهمة.

| النمط          | الصيغة               | النتيجة            |
| -------------- | --------------------- | ------------------- |
| عريض ومائل  | `**_عريض ومائل_**` | ***عريض ومائل*** |
| عريض ومشطوب     | `~~**عريض**~~`        | ~~**عريض**~~        |
| مائل ومشطوب | `~~*مائل*~~`      | ~~*مائل*~~      |

## القوائم

نظّم العناصر المرتبطة داخل تنسيق واضح وقابل للقراءة. يدعم Markdown القوائم غير المرتبة، والمرتبة، والمتداخلة لتلبية احتياجات مختلفة.

### غير مرتبة

استخدم القوائم غير المرتبة للعناصر التي لا تحتاج إلى ترتيب محدد. ابدأ كل عنصر بالرمز `-`.

::code-preview
---
class: "[&>div]:*:my-0"
---
- أنا عنصر في القائمة.
- أنا عنصر آخر في القائمة.
- أنا آخر عنصر في القائمة.

#code
```mdc
- أنا عنصر في القائمة.
- أنا عنصر آخر في القائمة.
- أنا آخر عنصر في القائمة.
```
::

### مرتبة

استخدم القوائم المرتبة عندما يكون ترتيب العناصر مهمًا، مثل خطوات عملية معينة. ابدأ كل عنصر برقم.

::code-preview
---
class: "[&>div]:*:my-0"
---
1. أنا عنصر في القائمة.
2. أنا عنصر آخر في القائمة.
3. أنا آخر عنصر في القائمة.

#code
```mdc
1. أنا عنصر في القائمة.
2. أنا عنصر آخر في القائمة.
3. أنا آخر عنصر في القائمة.
```
::

### متداخلة

أنشئ قوائم هرمية تحتوي على عناصر فرعية للبنى الأكثر تعقيدًا. ضع مسافة بادئة للعناصر الفرعية مقدارها أربع مسافات.

::code-preview
---
class: "[&>div]:*:my-0"
---
- أنا عنصر في القائمة.
  - أنا عنصر فرعي متداخل.
  - أنا عنصر فرعي متداخل آخر.
- أنا عنصر آخر في القائمة.

#code
```mdc
- أنا عنصر في القائمة.
  - أنا عنصر فرعي متداخل.
  - أنا عنصر فرعي متداخل آخر.
- أنا عنصر آخر في القائمة.
```
::

## الجداول

class: "[&>div]:*:my-0 [&>div]:*:w-full"

| ------- | --------- | -------- |
| `name`  |           | `string` |
| `size`  | `md`      | `string` |
| `color` | `neutral` | `string` |

## الاقتباسات

أبرز الاقتباسات أو المراجع أو النصوص المهمة. الاقتباسات تميّز المحتوى المقتبس بصريًا.

### سطر واحد

الاقتباسات ذات السطر الواحد مثالية للعبارات القصيرة والمؤثرة. أضف الرمز `>` قبل الفقرة.

::code-preview
---
class: "[&>div]:*:my-0"
---
> Nuxt UI هي مجموعة من مكونات Vue وcomposables وأدوات مساعدة مبنية على Nuxt UI، وموجهة للبنية والتخطيط، ومصممة كلبنات بناء لتطبيقك.

#code
```mdc
> Nuxt UI هي مجموعة من مكونات Vue وcomposables وأدوات مساعدة مبنية على Nuxt UI، وموجهة للبنية والتخطيط، ومصممة كلبنات بناء لتطبيقك.
```

::
