مكونات prose هي بدائل لوسوم الطباعة في HTML. وهي تمنحك طريقة بسيطة لتخصيص الواجهة أثناء استخدام Markdown.

يوفر **Docus وNuxt UI** مجموعة من مكونات prose الأنيقة والمصممة جيدًا لمساعدتك على كتابة التوثيق باستخدام [صيغة MDC](https://content.nuxt.com/docs/files/markdown#mdc-syntax).

::note{to="https://ui.nuxt.com/getting-started"}
تركّز هذه الصفحة فقط على مكونات prose الأكثر مناسبة لكتابة التوثيق. ومع ذلك، يمكنك استخدام **أي مكون من مكونات Nuxt UI** داخل Markdown. وللاطلاع على القائمة الكاملة للمكونات المتاحة، راجع توثيق Nuxt UI.
::

### `Callout`

استخدم Markdown داخل الفتحة الافتراضية لمكون `callout` لإضافة سياق بصري إلى المحتوى.

استخدم الخصائص `icon` و`color` لتخصيصه. يمكنك أيضًا تمرير أي خاصية من خصائص مكون `<NuxtLink>`.

يمكنك كذلك استخدام الاختصارات `note` و`tip` و`warning` و`caution` مع أيقونات وألوان محددة مسبقًا.

::tabs
  :::tabs-item{.my-5 icon="i-lucide-eye" label="معاينة"}
    ::::div{.flex.flex-col.gap-4.w-full}
      :::::note{.w-full.my-0}
      هذه معلومات إضافية لك.
      :::::

      :::::tip{.w-full.my-0}
      هذه نصيحة مفيدة.
      :::::

      :::::warning{.w-full.my-0}
      انتبه إلى هذا الإجراء لأنه قد ينتج عنه سلوك غير متوقع.
      :::::

      :::::caution{.w-full.my-0}
      هذا الإجراء لا يمكن التراجع عنه.
      :::::
    ::::
  :::

  :::tabs-item{icon="i-lucide-code" label="الكود"}
  ```mdc
  ::note
  هذه معلومات إضافية.
  ::

  ::tip
  هذه نصيحة مفيدة.
  ::

  ::warning
  انتبه إلى هذا الإجراء لأنه قد ينتج عنه سلوك غير متوقع.
  ::

  ::caution
  هذا الإجراء لا يمكن التراجع عنه.
  ::
  ```
  :::
::

### `Card` و`CardGroup`

استخدم Markdown داخل الفتحة الافتراضية لمكون `card` لإبراز المحتوى.

استخدم الخصائص `title` و`icon` و`color` لتخصيصه. يمكنك أيضًا تمرير أي خاصية من خصائص مكون `<NuxtLink>`.

اجمع مكونات `card` باستخدام مكون `card-group` لعرضها على هيئة شبكة.

::tabs
  :::tabs-item{.my-5 icon="i-lucide-eye" label="معاينة"}

      :::::card
      ---
      icon: i-simple-icons-github
      target: _blank
      title: لوحة التحكم
      to: https://github.com/nuxt-ui-templates/dashboard
      ---
      لوحة تحكم بتخطيط متعدد الأعمدة.
      :::::

      :::::card
      ---
      icon: i-simple-icons-github
      target: _blank
      title: SaaS
      to: https://github.com/nuxt-ui-templates/saas
      ---
      قالب يحتوي على صفحة هبوط وأسعار وتوثيق ومدونة.
      :::::

      :::::card
      ---
      icon: i-simple-icons-github
      target: _blank
      title: Docs
      to: https://github.com/nuxt-ui-templates/docs
      ---
      توثيق باستخدام `@nuxt/content`.

      :::::card
      ---
      icon: i-simple-icons-github

      title: Landing
      to: https://github.com/nuxt-ui-templates/landing
      ---
      صفحة هبوط يمكن استخدامها كنقطة انطلاق.

### `Collapsible`

غلّف المحتوى بمكون `collapsible` لعرض [Collapsible](https://ui.nuxt.com/components/collapsible) داخل المحتوى.

  :::tabs-item{.my-5 icon="i-lucide-eye" label="معاينة"}
    ::::collapsible
    | الخاصية    | الافتراضي    | النوع     |
    | ------- | --------- | -------- |
    | `name`  |           | `string` |
    | `size`  | `md`      | `string` |
    | `color` | `neutral` | `string` |
    ::::
  :::

  :::tabs-item{icon="i-lucide-code" label="الكود"}
  ```mdc
  ::collapsible

  | الخاصية    | الافتراضي    | النوع                     |
  |---------|-----------|--------------------------|
  | `name`  |           | `string`{lang="ts-type"} |
  | `size`  | `md`      | `string`{lang="ts-type"} |
  | `color` | `neutral` | `string`{lang="ts-type"} |

  ::
  ```

### `Field` و`FieldGroup`

`field` هو خاصية أو معامل تريد عرضه داخل المحتوى. ويمكنك تجميعها باستخدام `field-group` داخل قائمة.

::tabs
  :::tabs-item{.my-5 icon="i-lucide-eye" label="معاينة"}
    ::::field-group{.my-0}
      :::::field{name="analytics" type="boolean"}
      القيمة الافتراضية `false` - تفعّل التحليلات لمشروعك (قريبًا).
      :::::

      :::::field{name="blob" type="boolean"}
      القيمة الافتراضية `false` - تفعّل تخزين blob لحفظ الأصول الثابتة مثل الصور والفيديوهات وغيرها.
      :::::

      :::::field{name="cache" type="boolean"}
      القيمة الافتراضية `false` - تفعّل التخزين المؤقت لتخزين استجابات مسارات الخادم أو الدوال باستخدام `cachedEventHandler` و`cachedFunction` من Nitro.
      :::::

      :::::field{name="database" type="boolean"}
      القيمة الافتراضية `false` - تفعّل قاعدة بيانات SQL لتخزين بيانات تطبيقك.
      :::::
    ::::
  :::

  :::tabs-item{icon="i-lucide-code" label="الكود"}
  ```mdc
  ::field-group
    ::field{name="analytics" type="boolean"}
      القيمة الافتراضية `false` - تفعّل التحليلات لمشروعك (قريبًا).
    ::

  ::field{name="blob" type="boolean"}
    القيمة الافتراضية `false` - تفعّل تخزين blob لحفظ الأصول الثابتة مثل الصور والفيديوهات وغيرها.
  ::

  ::field{name="cache" type="boolean"}
    القيمة الافتراضية `false` - تفعّل التخزين المؤقت لتخزين استجابات مسارات الخادم أو الدوال باستخدام `cachedEventHandler` و`cachedFunction` من Nitro.
  ::

  ::field{name="database" type="boolean"}
    القيمة الافتراضية `false` - تفعّل قاعدة بيانات SQL لتخزين بيانات تطبيقك.
  ::
  ::
  ```

### `Steps`

غلّف العناوين بمكون Steps لعرض قائمة خطوات.

استخدم الخاصية `level` لتحديد مستوى العنوان الذي سيتم استخدامه كخطوات.

::tabs
  :::tabs-item{.my-5 icon="i-lucide-eye" label="معاينة"}
    ::::steps{level="4"}
    #### بدء مشروع جديد

    ```bash [Terminal]
    npm create nuxt@latest -- -t github:nuxt-content/docus
    ```

    #### تشغيل واجهة docus لبدء خادم التطوير

    ```bash [Terminal]
    docus dev
    ```
    ::::
  :::

  :::tabs-item{icon="i-lucide-code" label="الكود"}
  ````mdc
  ::steps{level="4"}
    #### بدء مشروع جديد
    
    ```bash [Terminal]
    npm create nuxt@latest -- -t github:nuxt-content/docus
    ```
    
    #### تشغيل واجهة docus لبدء خادم التطوير
    
    ```bash [Terminal]
    docus dev
    ```
  ::
  ````
