## واجهة الأوامر `create-docus`

::steps
### أنشئ مجلد التوثيق الخاص بك

استخدم واجهة الأوامر `create-docus` لإنشاء مشروع Docus جديد:

```bash [Terminal]
npx create-docus my-docs
```

يمكنك الاختيار بين قالبين:

- `default`: إعداد Docus أساسي لتوثيق بلغة واحدة
- `i18n`: يتضمن دعم التدويل لإنشاء توثيق متعدد اللغات

```bash [Terminal]
# الإنشاء باستخدام قالب i18n
npx create-docus my-docs -t i18n
```

نوصي باستخدام مدير الحزم `npm`.

### شغّل خادم التوثيق في وضع التطوير

انتقل إلى مجلد التوثيق الخاص بك ثم شغّل الخادم:

```bash [Terminal]
cd my-docs
npm run dev
```

ستتوفر معاينة التوثيق على <http://localhost:3000>

### اكتب توثيقك

انتقل إلى قسم [صيغة Markdown](/ar/essentials/markdown-syntax) لتتعلم طريقة كتابة التوثيق.
::

## دمج طبقة Docus

يستخدم Docus v4 **نهجًا قائمًا على طبقات Nuxt**، ويمكنك تمديد طبقة Docus مباشرة داخل ملف `nuxt.config.ts` باستخدام `extends: ['docus']`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  extends: ['docus']
})
```
