#features
  :::u-page-feature
  ---
  icon: i-simple-icons-nuxt
  target: _blank
  to: https://nuxt.com
  ---
  #title
  مبني باستخدام [Nuxt 4]{.text-primary}
  
  #description
  مدعوم بإطار Vue المفضل لديك. يمنحك Docus كل ما تحتاجه لإنشاء مواقع سريعة وعالية الأداء ومحسّنة لمحركات البحث.
  :::

  :::u-page-feature
  ---
  icon: i-simple-icons-nuxt
  target: _blank
  to: https://ui.nuxt.com/
  ---
  #title
  مصمم بواسطة [Nuxt UI]{.text-primary}
  
  #description
  أنيق، بسيط، وقابل للتخصيص. يدمج Docus مع Nuxt UI ليمنحك أفضل تجربة لكتابة التوثيق بدون تعقيد أو قوالب زائدة، فقط ركّز على محتواك.
  :::

  :::u-page-feature
  ---
  icon: i-simple-icons-nuxt
  target: _blank
  to: https://content.nuxt.com
  ---
  #title
  Markdown محسّن بواسطة [Nuxt Content]{.text-primary}
  
  #description
  الشيء الوحيد الذي تحتاج إلى الاهتمام به هو كتابة المحتوى. اكتب صفحاتك بصيغة Markdown وادمج مكونات Nuxt UI أو مكونات Vue مخصصة. تتم إدارة البنية، والمسارات، والتصيير نيابةً عنك.
  :::

  :::u-page-feature
  ---
  icon: i-simple-icons-nuxt
  target: _blank
  to: https://nuxt.com/docs/guide/directory-structure/app-config
  ---
  #title
  تخصيص عبر [Nuxt App Config]{.text-primary}
  
  #description
  حدّث الألوان والروابط الاجتماعية والشعارات وحتى أسلوب المكونات عالميًا عبر `app.config.ts` بدون تعديل مباشر في الكود.
  :::

  :::u-page-feature
  ---
  icon: i-simple-icons-nuxt
  target: _blank
  to: https://content.nuxt.com/studio
  ---
  #title
  حرر المحتوى في الإنتاج باستخدام [Nuxt Studio]{.text-primary}
  
  #description
  عدّل محتواك في بيئة الإنتاج بدون الحاجة إلى أي معرفة بـ Markdown. اسمح لزملائك غير التقنيين بالتعاون على التوثيق ودمج مكونات Vue بدون مهارات برمجية.
  :::

  #description

  :::
