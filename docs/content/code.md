## Simple Counter with Preview

Counter example using `<CodePreview>` component

<CodePreview :preview="true">

<CodeGroup>

```vue [App.vue]
<template>
    <MyCounter />
</template>

<script>
import MyCounter from './MyCounter.vue'
export default {
    components: { MyCounter }
}
</script>
```

```vue [MyCounter.vue]
<template>
    <div>
        <div>
            Count: <strong>{{ count }}</strong>
        </div>
        <MyButton @click="increaseCount" title="Click Me" />
    </div>
</template>

<script>
import MyButton from './MyButton.vue'
export default {
    components: { MyButton },
    data () {
        return {
            count: 0
        }
    },
    methods: {
        increaseCount() { this.count += 1 }
    }
}
</script>
```

```vue [MyButton.vue]
<template>
    <button @click="$emit('click')">
        <slot>
            {{ title }}
        </slot>
    </button>
</template>
<script>
export default {
    props: {
        title: {
            type: String,
            default: ""
        }
    }
}
</script>
```
